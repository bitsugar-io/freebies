# App Store Screenshots

Capture on iPhone 15 Pro Max simulator (6.7" - 1290x2796). App Store Connect will scale for smaller
devices.

## How to Capture

1. Run app in iPhone 15 Pro Max simulator
2. Press `Cmd + S` to save screenshot
3. Rename and move to this folder

## Tips

- Use light mode for consistency
- Have realistic data visible (multiple leagues, deals)
- For notification screenshot, you can mock it or use a notification preview tool
- Make sure status bar shows good signal/battery
